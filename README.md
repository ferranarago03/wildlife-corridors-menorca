# Menorca Ecological Corridor Optimization

This project implements advanced mathematical optimization models to design ecological corridors for wildlife in Menorca. It uses **Gurobi** and **Google OR-Tools** to solve multi-species connectivity problems, balancing conservation goals with economic constraints.

Clone the repository and install dependencies using `uv`:

```bash
git clone git@github.com:ferranarago03/wildlife-corridors-menorca.git
```

The **Direct Multi-Species Model** is the main production model:

```bash
uv run src/models/modelling_multi_specie.py
```
